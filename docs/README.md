# docs

Teaching pages and "build one, document the rest" comparisons. Each page explains a concept
as implemented in Groundwork, then surveys the alternatives the brief asks about.

Planned pages (filled in alongside the phase that builds the concept):

- `agents.md` - LangGraph vs AutoGen vs CrewAI
- `retrieval.md` - hybrid search; Chroma vs Pinecone / Weaviate / Milvus
- `mcp.md` - what MCP is and why a protocol boundary beats in-process tools
- `eval.md` - evaluating foundation models; the cost / latency / accuracy frontier
- `guardrails.md` - deterministic safety controls and where they sit
- `observability.md` - tracing reasoning chains with OpenTelemetry
- `cost.md` - prompt + semantic caching and token-cost optimization
- `cloud.md` - AWS deployment; Azure / GCP comparison
